# Website Color Scheme

## Brand Colors

### Primary Color - Coral/Orange
The main brand color is a vibrant coral/orange that represents energy, innovation, and transformation.

- **Primary 500 (Main)**: `#FF6B4D` - Main brand color
- **Primary 600**: `#FF4D2B` - Darker variant
- **Primary 700**: `#E63D1A` - Even darker
- **Primary 400**: `#FF8D7A` - Lighter variant
- **Primary 300**: `#FFB5A8` - Light variant
- **Primary 200**: `#FFD1C9` - Very light
- **Primary 100**: `#FFE8E4` - Subtle background
- **Primary 50**: `#FFF4F2` - Lightest background

**Usage**: Primary CTAs, buttons, links, highlights, active states

### Secondary Color - Navy Blue
A deep navy blue that conveys professionalism, trust, and stability.

- **Secondary 900 (Main)**: `#030F35` - Main secondary color
- **Secondary 800**: `#141F42` - Slightly lighter
- **Secondary 700**: `#1F2F5C` - Medium dark
- **Secondary 600**: `#2E4580` - Medium
- **Secondary 500**: `#4A5FA8` - Lighter variant
- **Secondary 400**: `#6B7DBF` - Light variant
- **Secondary 300**: `#9BA8D6` - Very light
- **Secondary 200**: `#C3CBE7` - Subtle
- **Secondary 100**: `#E1E5F3` - Background
- **Secondary 50**: `#F0F2F9` - Lightest background

**Usage**: Headers, text, backgrounds, navigation, footer

## Accent Colors

### Accent (Coral)
- **Default**: `#FF6B4D` (same as primary)
- **Light**: `#FF8D7A`
- **Dark**: `#E63D1A`

**Usage**: Emphasis, hover states, important UI elements

### Teal
- **Default**: `#00D4C5`
- **Dark**: `#00A89C`
- **Light**: `#66E8DE`

**Usage**: Success states, complementary accents, data visualization

### Purple
- **Default**: `#8B5CF6`
- **Dark**: `#6D28D9`
- **Light**: `#A78BFA`

**Usage**: Premium features, complementary accents, data visualization

### Pink
Pink is used extensively throughout the website for various UI elements:
- **Pink 50**: Very light pink backgrounds
- **Pink 100**: Light pink backgrounds for badges
- **Pink 500**: Medium pink for indicators and dots
- **Pink 600**: Darker pink for gradients and buttons
- **Pink 700**: Dark pink for hover states
- **Pink 800**: Very dark pink for text

**Usage**: 
- High impact badges and labels
- Gradient backgrounds (combined with purple)
- Podcast content type indicators
- Milestone markers
- CTA backgrounds
- Sector landing page gradients (Experience 4.0, Retail)
- Infographic media type buttons

## Typography

### Font Families
- **Display/Headings**: Plus Jakarta Sans (sans-serif)
- **Body Text**: DM Sans (sans-serif)

### Font Usage
- `font-display` or `font-serif`: Headlines and major headings
- `font-body` or `font-sans`: Body text, paragraphs, UI elements

## Color Palette Summary

### Primary Palette
1. **Coral Orange** (`#FF6B4D`) - Main brand color for CTAs and highlights
2. **Navy Blue** (`#030F35`) - Secondary brand color for text and backgrounds

### Supporting Colors
3. **Teal** (`#00D4C5`) - Complementary accent
4. **Purple** (`#8B5CF6`) - Complementary accent
5. **Pink** (Tailwind default pink scale) - High impact indicators, gradients, badges

### Utility Colors
The website also uses standard Tailwind colors for various UI states:
- **Blue**: Information, links
- **Green**: Success states
- **Yellow**: Warnings
- **Red**: Errors, alerts
- **Gray**: Neutral elements, borders, disabled states

## Dark Mode Support
The website supports dark mode via the `class` strategy. Colors automatically adapt based on the `dark:` prefix in Tailwind classes.

## Design System Notes
- The color scheme emphasizes warmth (coral) balanced with professionalism (navy)
- High contrast between primary and secondary colors ensures accessibility
- Teal and purple provide variety for data visualization and UI differentiation
- The extended color scales (50-900) allow for subtle variations and depth
