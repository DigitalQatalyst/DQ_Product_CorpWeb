# DigitalQatalyst - Digital Experience Platform (DXP)

A comprehensive digital transformation platform for DigitalQatalyst, featuring marketplaces, client portal, content management, and CRM integration.

## Brand Colors

- **Primary (Accent)**: `#FF6B4D` - Main CTA buttons, highlights, accents
- **Secondary**: `#030F35` - Surfaces, secondary buttons, text

## Getting Started

1. Run `npm install`
2. Run `npm run dev`

## Platform Architecture

### Stage 00 - Landing Page
Corporate homepage showcasing DQ's digital transformation services

### Stage 01 - Marketplaces
- Services Marketplace
- Products Marketplace
- Knowledge Hub (Articles, Videos, Case Studies)
- Resources & Tools

### Stage 02 - Transaction
- Service inquiry/booking forms
- Lead capture and qualification
- Client onboarding

### Stage 03 - CRM/Client Portal
- Authenticated client dashboard
- Service order tracking
- Exclusive content access
- Document management
- Support ticketing

## Auth Pattern

- See `docs/auth-config-pattern.md` for the MSAL-based authentication configuration pattern, env variables, and integration details.
