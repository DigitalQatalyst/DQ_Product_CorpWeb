## Story
DigitalQatalyst is the definitive digital transformation consultancy that bridges the gap between strategic vision and tactical execution. The platform serves multiple audiences simultaneously: C-suite executives seeking strategic guidance, transformation teams needing tactical support, and digital workers requiring capability development. The website positions DQ as the trusted advisor through thought leadership, proven methodologies, and demonstrated results.

Beyond a simple corporate website, the DQ platform is a comprehensive digital experience that combines public storytelling, marketplace discovery, sophisticated lead qualification, and authenticated client portals. The experience showcases the 6 Dimensions of Digital (6XD) framework, highlights proven methodologies, and provides seamless pathways from initial interest to long-term partnership.

## Purpose
Provide DigitalQatalyst's corporate presence, thought leadership platform, services marketplace, lead conversion engine, client portal, and operational intelligence across the entire client lifecycle.

## Technical Architecture

### Frontend Stack
- **Framework**: React 19.2 with TypeScript
- **Build Tool**: Vite 7.1 for fast development and optimized builds
- **Styling**: Tailwind CSS with Mantine UI components
- **State Management**: React Context API with custom hooks
- **Routing**: React Router v6 with protected routes
- **Forms**: React Hook Form with Yup validation
- **Rich Text**: TipTap editor with custom extensions
- **Charts**: Recharts for analytics visualization
- **Maps**: Leaflet with React Leaflet integration

### Backend & Services
- **Authentication**: Azure B2C with MSAL integration
- **Database**: Supabase PostgreSQL with Row Level Security
- **CRM Integration**: Airtable for lead management and form submissions
- **AI Services**: Azure OpenAI for digital maturity assessments
- **File Storage**: Azure Blob Storage for media assets
- **Email**: Custom email service integration
- **Analytics**: Google Analytics 4 with custom event tracking

### Infrastructure & DevOps
- **Hosting**: Vercel for frontend deployment
- **CDN**: Vercel Edge Network for global content delivery
- **Monitoring**: Built-in error tracking and performance monitoring
- **Security**: HTTPS, CSP headers, secure authentication flows
- **Backup**: Automated database backups and disaster recovery
